BOLTS

BOLTS ist eine offene Bibliothek für technische Spezifikationen.

Was ist BOLTS?

Das Ziel von BOLTS ist es, eine freie und quelloffene Standardteilebibliothek für CAD Applikationen aufzubauen. BOLTS tut dies, indem es eine gemeinsame Datenbasis bereitstellt, die von verschiedenen CAD Programmen aus genutzt werden kann.

Was ist eine Standardteilebibliothek?

Wenn man eine CAD Software verwendet um ein Objekt zu konstruieren, dann besteht dieses normalerweise aus zwei Sorten von Komponenten: Spezialteile, die spezifisch für diese Anwendung sind, und die selbst hergestellt werden müssen (z.B. mit einem 3D Drucker), und Standardteile, wie Schrauben und Muttern, die man im Laden kaufen kann.

Eine Standardteilebibliothek nimmt einem das langwierige Modellieren von Standardteilen ab, indem es diese fertig nutzbar bereitstellt. Dadurch kann man sich auf die interessanten Aspekte eines Designs konzentrieren.

Während viele Firmen Standardteilebibliotheken für kommerzielle Software anbieten, gibt es bisher keine, die frei ist (wie in Freiheit).

Wie funktioniert BOLTS?

BOLTS nutzt die parametrischen Funktionen, die die meisten CAD Programme bereitstellen. Das erlaubt es, eine große Zahl von verschiedenen Größen und Varianten von einem einzelnen Modell abzuleiten.

Abmessungen und Metadaten für alle Größen und Versionen eines Teils sind in speziellen Dateien verfügbar. BOLTS can diese Daten zusammen mit den parametrischen Modellen dazu nutzen, automatisch Standardteilebibliotheken (\"BOLTS für ...\") verschiedene CAD programme zu erstellen, die der Benutzer einfach herunterladen und verwenden kann.

BOLTS ist also in einem gewissen Sinne ein Werkzeug zur Erstellung von Standardteilebibliotheken.

Wie kann ich mithelfen?

Du kannst bei BOLTS mithelfen, indem du Teile in einem der unterstützten CAD Programmen modellierst, Abmessungen und Metadaten für neue Teile hinzufügst oder Zeichnungen für das Teileverzeichnis erstellst.

Diese Aufgaben sind relativ einfach, und die meisten erfordern keine Programmierkenntnisse. Um mehr zu erfahren, besuche diese Seite mit weiteren Informationen und Links zu detaillierten Beschreibungen der verschiedenen Aufgaben.

Schnelleinstieg: BOLTS für OpenSCAD verwenden

Laden Sie die komprimierte Datei mit BOLTS für OpenSCAD von der Downloadseite herunter und entpacken Sie es in einen Ordner. Starten Sie OpenSCAD von diesem Ordner aus oder passen Sie ihren OpenSCAD Bibliothekspfad an um sicherzustellen, dass OpenSCAD BOLTS auch finden kann. Dann fügen sie diese Zeile am Anfang Ihrer scad Datei ein:


include <bolts.scad>

Jetzt können Die BOLTS verwenden um Standardteile wie eine Unterlagsscheibe einzufügen:


ISO7089("M4");

Schnelleinstieg: BOLTS für FreeCAD verwenden

Laden Sie die komprimierte Datei mit BOLTS für FreeCAD von der Download Seite herunter und extrahieren Sie ihren Inhalt in das FreeCAD Macroverzeichnis. Falls sie nicht wissen, welches Verzeichnis das ist, können sie den Pfad unter dem Menüpunkt Eigenschaften im Reiter "Macro" nachschauen. Sie müssen ausserdem sicherstellen, dass Sie pyyaml installiert haben.

Nachdem sie FreeCAD gestartet haben, wählen sie im Menü Macro den Menüpunkt Macros aus und führen das Macro start_bolts.FCMacro aus

Ein Fenster sollte erscheinen. In diesem können sie ein Teil aus einem Baum auswählen, Werte für die Parameter festlegen und das Teil zum aktuellen Dokument hinzufügen.

Für weitere Informationen konsultieren Sie bitte die Dokumentation

Schnelleinstieg: BOLTS für IGES verwenden

Laden Sie die komprimierte Datei mit BOLTS für IGES von der Downloadseite herunter und entpacken Sie sie in einen Ordner. Das kann mehrere hundert MByte an Platz erfordern.

IGES ist ein weitverbreitetes Format, so dass fast jedes CAD Programm es unterstützt. Die Details hängen vom jeweiligen Programm ab, aber üblicherweise muss man eine IGES Datei öffnen oder importieren um das darin enthaltene Teil nutzen zu können.

Die Dateien sind in Unterverzeichnissen organissiert, die den Kollektionen entsprechen, die in der Teileliste aufgeführt sind. Nicht alle Größen und Varianten aller Teile sind in BOLTS für IGES verfügbar, wenn Ihnen etwas fehlt, machen Sie mit.
