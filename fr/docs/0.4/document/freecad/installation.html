Installer BOLTS pour FreeCAD

Télécharger BOLTS pour FreeCAD dernière version

On peut trouver la dernière version de BOLTS pour FreeCAD sur la Page de téléchargement.

Différentes types d'archives sont disponibles. Si vous ne savez pas lequel choisir, prendre le zip.

Extraire l'archive dans le dossier des macros

On peut voir la localisation du dossier des macros dans les préférences: Choisir Édition->Préférences, dans la fenêtre sélectionner l'onglet macro. Le chemin vers le dossier des macro y est indiqué. Sous linux le dossier des macros est souvent situé à ~/.FreeCAD.

Une fois fait le dossier macro devrait contenir un sous-répertoire nommé BOLTS et un fichier nommé start_bolts.FCMacro

L'installation est maintenant terminé.

Essaye-le

Démarrer FreeCAD et choisir Macros dans le menu Macro. Dans la fenêtre qui apparaît sélectionner start_bolts.FCMacro et cliquer sur exécuter.

Si vous avez un erreur disant quelque chose comme ceci :

Pas de module nommé yaml

alors c'est que la bibliothèque yaml pour python n'est pas installé. si vous utilisez linux, vous pouvez souvent l'obtenir a partir du gestionnaire de paquet. si vous↵ utilisez windows vous pouvez récupérer l'installateur ici; vous↵ devez installer YAML pour la version de Python (les deux premier chiffres) qui est↵ distribué avec FreeCAD. Vous pouvez retrouver quelle version est utilisé à la première↵ ligne de la console python de FreeCAD. Si elle n'est pas ouverte, vous pouvez↵ la faire apparaître en la cochant dans le Menu Affichage dans Vues.

Si vous avez un erreur disant quelque chose comme ceci :

Pas de module nommé importlib

C'est parce que vous utilisez une ancienne version de python ne comportant pas la bibliothèque importlib. Vous pouvez la récupérer ici. Il n'y a pas d'installateur d'importlib disponible spécialement pour windows. Les instructions↵ pour installer un module python qui ne possède pas d’installateur est donné↵ ici. Encore une fois vous devez utiliser la même version de python que celle distribué avec FreeCAD.

Si vous obtenez un message disant

l'importation uic a échoué. Vérifier que l'outil pyside est installé

alors c'est qu'il manque un élément de PySide pour la liaison Qt. Pour Debian et Ubuntu↵ il est contenu dans le paquet pyside-tools.

Si vous avez d'autres erreurs, essayez la dernière version de développement. Si ca ne fonctionne toujours pas, merci de rapporter le problème. Le plus simple est d'utiliser les commentaires si dessous, mais il y a de nombreux autres endroits ou vous pouvez rapporter un problème, une liste se trouve↵ sur la page Contribuer.
