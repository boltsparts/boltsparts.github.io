Documentation for BOLTS 0.3 (current stable version)

stable dev

Créer un bouton pour BOLTS dans la barre d'outils

Ceci est une version adapté et étendu de la description du wiki de FreeCAD.

Vous avez besoin d'avoir installé BOLTS pour FreeCAD.

Ouvrir la boîte de dialogue Personnaliser

On peut la trouver dans Outils->Personnaliser. Sélectionner l'onglet Macros. Si vous obtenez un message d'erreur vérifier que vous avez BOLTS pour FreeCAD installé.

Sélectionnez la macro et remplir les champs

Sélectionner start_bolts.FCMacro depuis la liste du menu-déroulant . Les différents champs peuvent être remplis par Démarrer BOLTS bibliothèque de pièces standard ou quelque chose de similaire.

Puis cliquer sur les trois points a coté de Pixmap, choisir Dossier d'icônes et ajouter le dossier de BOLTS situé dans votre répertoire de macro (ceci peut s’avérer compliqué étant donné que le dossier de macro est caché). Ajouter les icônes du dossier en cliquant sur Ouvrir. Enfin choisissez l'icône doré en forme de boulon nommée bolts32.png.

Lorsque vous avez terminé, ça devrait ressembler à ceci :

Confirmez la macro personnalisée

Vous faites cela en cliquant sur le bouton Add.

Si vous souhaitez changer quelque chose à posteriori, vous pouvez sélectionner la macro personnalisé dans la liste sur la gauche en double-cliquant, faîte vos changement, puis confirmer en cliquant sur le bouton Replace.

Créer une nouvelle barre d'outils et ajouter le bouton

Dans la boîte de dialogue Personnaliser sélectionner l'onglet Barre d'outils. Dans la liste du menu déroulant de gauche sélectionner Macro, sur la droite sélectionner l'Atelier dans lequel vous souhaitez créer le bouton de la barre d'outils.

Puis créer une nouvelle barre d'outils en cliquant 'Nouveau...' et la nommer BOLTS. La macro de démarrage peut être ajouté en la sélectionnant dans la liste de gauche puis en cliquant sur la flèche vers la droite.

Lorsque vous avez terminé, ça devrait ressembler à ceci :

Puis fermer la boîte de dialogue Personnaliser.

Essaye-le

Passez à l'Atelier auquel vous avez ajouté la barre d'outils à l'étape précédente. Ou changer puis revenir si cette barre d'outils est actuellement active.

Vous devriez maintenant avoir une nouvelle barre d'outils, et un clic sur l'icône jaune en forme de boulon doit ouvrir le sélecteur de pièces standard BOLTS.
