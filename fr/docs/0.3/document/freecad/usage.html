Documentation for BOLTS 0.3 (current stable version)

stable dev

Utiliser BOLTS pour FreeCAD

Vous avez besoin d'avoir installé BOLTS pour FreeCAD.

Démarrer le sélecteur de pièce avec une macro

Choisir Macros depuis le menu Macro et executer start_bolts.FCMacro.

Démarrer le sélecteur de pièce avec le bouton dans la barre d'outils

Si vous avez défini un bouton dans la barre d'outils, vous pouvez démarrer le sélecteur de pièces en cliquant simplement sur le bouton

Les différents éléments du sélecteur de pièces

Les pièces disponibles sont triées de deux façons :

  1. thématiquement par collections
  2. par l'organisme de normalisation si normalisé

En développant arborescence on peut naviguer dans les pièces disponibles. Dans la zone entre l’arborescence et le bouton s'affiche les informations sur la sélection en cours.

Vous pouvez voir que certaines pièces (comme DIN931, DIN933) sont visible deux fois dans cette capture d'écran, une fois dans la collection hexagon fastener, une fois dans la liste du standard DIN.

Spécifier les paramètres et insérer la pièce

Si vous sélectionnez une pièce, par exemple ISO4017, un autre hexagon fastener, en dessous de la zone d'informations des éléments complémentaires apparaissent qui permettent de donner les valeurs aux paramètres de la pièces pas encore spécifiée.

Pour connaître le sens des paramètres, on peut consulter la page de spécification BOLTS pour cette pièce. La navigation au seins des pages de spécification est structuré de la même façon que celle dans le sélecteur de pièces.

Dans le cas d'ISO4017, on a besoin de donner une longueur et un diamètre. Le diamètre peut être choisi depuis le menu-liste déroulant qui contient l'ensemble des choix connu de BOLTS, la longueur peut être entré dans la ligne d'édition. Il accepte seulement des valeurs positives.

Une fois les paramètres spécifiés, La pièce peut être insérée dans le document actif en cliquant sur le bouton Add part. Dans la vue combo sur le coté droit vous pouvez voir que le label de l’objet donne le standard et la valeur de tout les paramètres que l'on a spécifié.
